# ChatPwa-4IWJ X Firebase

## Initilisation projet
- Déplacer dans app
```
cd app
```
- Dupliquer .env.example et renommer-le en .env
- Remplir le nouveau .env avec les informations nécessaires
- Installer les dépendances 
```
npm install
```

## Lancer projet
- Exécuter la commande
```
npm start
```
